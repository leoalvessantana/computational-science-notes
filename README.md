# Computational Science Notes

An open-source, continuously growing collection of notes on Computational Science — written to be clear, intuitive, and rigorous, prioritizing understanding before formalization.

📖 **Read it here:** https://computational-science-notes.readthedocs.io/en/latest/

## About

This project started as a personal knowledge base and is now maintained as a long-term, evolving reference. Rather than a static textbook, it's meant to grow alongside my own studies — each topic is written the way I wish I'd found it when I first started learning it.

## Topics Covered

The notes are organized into thematic sections, currently including:

- **Quantum Computing** *(in progress)* — mathematical foundations, qubits, quantum gates, entanglement, quantum algorithms, and quantum machine learning.
